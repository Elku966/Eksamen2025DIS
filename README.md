
Tilgå vores app via Naee.dev, hvis den ønskes startet på localhost gør følgende: 

1. Kør npm install i terminallen for at installere nødvendige moduler 

2. Opret en .env fil med dine nøgler: 
    TWILIO_ACCOUNT_SID=din_twilio_account_sid
    TWILIO_AUTH_TOKEN=dit_twilio_auth_token
    TWILIO_PHONE_NUMBER=dit_twilio_telefonnummer

3. Skriv npm start i terminalen
